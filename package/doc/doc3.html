设立人保交通管理服务站



随着我市经济社会的迅速发展，机动车保有量急速增长，截至2015年年底，我市机动车保有量已超过86万余辆，机动车驾驶员大量增加，汽车进入百姓家庭，带动了群众生活品质的提高，对交通管理的需求也有了新的发展，特别是“车、驾”管业务，已经成为当前重要的民生问题。绵阳市车管所与中国人保现有的服务网点有效结合，委托人保代办“车、驾”管业务服务点，共同提高服务效率，方便群众，惠及民生，同时也对减少和缓解城市交通压力起着重大作用。




服务站业务范围



驾驶证管理业务：

  1. 驾驶证期满换证、遗失补领、损毁换领；
  2. 达到规定年龄换领、信息变更换领、信息变更备案；
  3. 提交身体条件证明；
  4. 变更联系方式备案；

车辆管理业务范围：

  1. 机动车号牌遗失补领，损毁换领；
  2. 行驶证遗失补领，损毁换领；
  3. 检验合格标志遗失补领，损毁换领；
  4. 机动车所有人住所地址变更换证；
  5. 机动车所有人联系方式变更备案；
  6. 免检车辆核发检验合格标志、免收代办服务费；

本服务代办点为合规的代办点、所提供的各项服务全部免费，仅按照车管所的收费标准来收取工本费，并出据合规的“四川省政府非税收收入通用票据”。
